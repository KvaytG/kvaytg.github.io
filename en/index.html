win-tools

Challenge: Accessing native Windows APIs from Java applications.

Solution: A wrapper library over custom DLLs that provides a Java interface for system operations: mouse and terminal control, notifications, dialog boxes, and critical functions (BSOD, shutdown).

Source Code (GitHub)

ru-chat-bot

Challenge: Developing an intelligent chatbot with Russian Natural Language Understanding.

Solution: A modular AI bot powered by deep learning. It utilizes Transformers (RuBERT) for vectorization, an NLU model for intent classification, NER for entity extraction, and FAISS for semantic search. Supports weather, news, Wikipedia search, and smart dialogue.

Source Code (GitHub)

ru-toxicity-detector

Challenge: Automated detection of toxic content in Russian text.

Solution: A hybrid model combining neural network embeddings (rubert-tiny2) with a dictionary-based approach. Optimized for high precision (98%) with minimal false positives. Includes an exception system for context-aware classification.

Source Code (GitHub)

ru-wiki-search

Challenge: Fast local search through Russian Wikipedia titles with fuzzy matching support.

Solution: A library featuring an SQLite FTS5 index, utilizing lemmatization and combined ranking metrics (fuzzy matching + semantic similarity). Automatically downloads and processes Wikipedia dumps, providing brief article descriptions.

Source Code (GitHub)

manhwa-russifier

Challenge: Automated localization for manhwa.

Solution: A complete processing pipeline: text area detection via YOLOv8 → EasyOCR recognition → translation → original text removal using LaMa inpainting → rendering translated text with context awareness.

Source Code (GitHub)

fb2-converter

Challenge: Converting various text formats into the FictionBook 2.0 standard.

Solution: A tool for transforming TXT, PDF, and EPUB files into FB2 while preserving structure, generating covers and tables of contents, and auto-detecting headers.

Source Code (GitHub)

en-ru-corpus-utils

Challenge: Preparing high-quality parallel corpora for machine translation.

Solution: A set of utilities for cleaning, filtering, and validating English-Russian text pairs using statistical, linguistic, and semantic (LaBSE) filtering methods.

Source Code (GitHub)

core-mc-plugin

Challenge: Comprehensive management and customization for Minecraft servers on Paper 1.16.5.

Solution: A multi-functional plugin combining warp systems, vanish mode, moderation tools, player perks, and restrictions with configurable permissions and localization.

Source Code (GitHub)

discord-music-bot

Challenge: Playing music in Discord with full player control.

Solution: A Java-based bot with modular architecture, providing slash commands for playback management, queueing, volume control, and an interactive player supporting Bandcamp, Vimeo, SoundCloud, and other platforms.

Source Code (GitHub)

caption-forge

Challenge: Automatic text overlay on images with adaptive formatting.

Solution: A library for automatic font size selection, line wrapping, text coloring, and outlining to ensure optimal readability on any background. Uses color analysis and ROI geometry algorithms.

Source Code (GitHub)

simple-unzipper

Challenge: Secure and convenient ZIP archive extraction via command line.

Solution: A Java CLI application featuring protection against ZipSlip attacks, unique folder naming to avoid conflicts, and cross-platform output (colored terminal + Windows notifications).

Source Code (GitHub)
